# Customer Health & Churn Prediction Copilot

A machine learning-powered customer success dashboard built with Python, Streamlit, Pandas, and Scikit-Learn.

This application analyzes customer engagement, adoption, support, and satisfaction metrics to predict churn risk and generate customer health insights. The solution provides churn probability scoring, health status classification, recommended customer success actions, and interactive dashboards to help identify at-risk accounts.

## Features

* Customer CSV upload and analysis
* Churn prediction using Random Forest Classifier
* Customer health scoring and segmentation
* Recommended next-best actions for Customer Success Managers
* Interactive dashboard and visualizations
* Risk ranking and customer-level insights

## Technology Stack

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-Learn
* Random Forest Machine Learning Model

## Business Value

This project demonstrates how machine learning can be applied to customer success workflows to proactively identify churn risk, prioritize account management efforts, and improve customer retention outcomes.
